COPD Prediction System

Code Live Demo

Project Overview

A comprehensive web application designed to assess users' risk of Chronic Obstructive Pulmonary Disease (COPD) based on various health parameters and lifestyle factors.

Key Features

  • Machine learning model trained on clinical data to predict COPD risk
  • User-friendly interface for inputting health metrics
  • Automated connection to healthcare professionals when high risk is detected
  • Educational resources about COPD prevention and management

Technologies Used

Python Flask Scikit-learn JavaScript HTML/CSS

Project Impact

This system has the potential to enable early detection of COPD, which is crucial for effective management of the disease. The proactive healthcare connection feature bridges the gap between risk assessment and professional medical consultation.
